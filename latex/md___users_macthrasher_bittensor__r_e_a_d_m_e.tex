This repository contains Bittensor\textquotesingle{}s Python API, which can be used for the following purposes\+:


\begin{DoxyEnumerate}
\item Querying the Bittensor network as a \href{https://github.com/opentensor/bittensor\#31-client}{\texttt{ client}}.
\item Running and building Bittensor miners and validators for \href{https://github.com/opentensor/bittensor\#43-running-a-template-miner}{\texttt{ mining TAO}}.
\item Pulling network \href{https://github.com/opentensor/bittensor\#3-using-bittensor}{\texttt{ state information}}.
\item Managing \href{https://github.com/opentensor/bittensor\#41-cli}{\texttt{ TAO wallets}}, balances, transfers, etc.
\end{DoxyEnumerate}

Bittensor is a mining network, similar to Bitcoin, that includes built-\/in incentives designed to encourage miners to provide value by hosting trained or training machine learning models. These models can be queried by clients seeking inference over inputs, such as token-\/based text generations or numerical embeddings from a large foundation model like GPT-\/\+Neo\+X-\/20B.

Token-\/based incentives are designed to drive the network\textquotesingle{}s growth and distribute the value generated by the network directly to the individuals producing that value, without intermediaries. The network is open to all participants, and no individual or group has full control over what is learned, who can profit from it, or who can access it.

To learn more about Bittensor, please read our \href{https://drive.google.com/file/d/1VnsobL6lIAAqcA1_Tbm8AYIQscfJV4KU/view}{\texttt{ paper}}.
